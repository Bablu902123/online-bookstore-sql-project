# Online Bookstore SQL Project

## Project Overview
This project demonstrates SQL skills by analyzing an online bookstore database.  
It includes database creation, data import from CSV files, and solving real-world business queries.

## Database Tables
Books
Customers
Orders

## Skills Demonstrated
SQL Joins
Aggregation Functions
Group By & Having
Filtering Data
Data Analysis Queries

## Key Analysis Performed
Total revenue calculation
Most frequently ordered book
Customer spending analysis
Stock remaining after orders
Books sold per genre

## Tools Used
PostgreSQL
SQL
CSV Dataset
